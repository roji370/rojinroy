Course Recommendation Model

Python Scikit-Learn Pandas Flask API MongoDB

Project Overview

The Course Recommendation Model is an intelligent system built during my undergrad studies to solve a common problem in academic environments: curriculum overload and decision fatigue. By analyzing student performance metrics, individual career aspirations, and historic enrollment statistics, the system recommends optimal course selections tailored for each student.

Key Features

  • Content-Based Filtering: Recommends courses matching students' historical interest profile.
  • Collaborative Filtering: Leverages peer performance profiles to recommend courses where students with similar profiles excelled.
  • Prerequisite Validator: Built-in graph algorithm to check and warn students of missing prerequisites in real-time.
  • API Integration: Exposes recommendations via a lightweight Flask JSON API.

Technical Execution

The core ML pipeline is built in Python using Scikit-Learn for vectorization (TF-IDF on course syllabus data) and Cosine Similarity computations. We utilized Pandas and NumPy for dataset ingestion and cleaning. The system utilizes MongoDB to store course metadata, user logs, and enrollment vectors.
