Cross-Platform Mobile Showcase

Flutter React Native Dart / JS Firebase REST API Sync

Project Overview

As a freelance developer, having a high-performance cross-platform template is essential. This **Mobile Showcase** represents a prototype fitness tracking application built concurrently using **Flutter** and **React Native** to compare engine rendering, bridge responsiveness, and compilation speeds.

Key Features

  • Offline-First Syncing: Local SQLite cache that automatically reconciles data vectors with MongoDB once a network is restored.
  • Asynchronous State Management: Leveraged BLoC (Flutter) and Redux-Toolkit (React Native) for fluid UX transitions.
  • Biometric Security: TouchID/FaceID hardware integrations for user session locks.
  • Dynamic Charts: Custom render canvas displaying user activity metrics.

Technical Specifications

Both codebases share a unified RESTful backend API written in Node/Express and backed by MongoDB. Testing cycles focused heavily on CPU profiling, frame drops, and battery consumption rates across target iOS and Android simulator platforms.
